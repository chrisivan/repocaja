# Proyecto colaborativo del curso de Git/GitHub

Este repositorio es utilizado para la práctica colaborativa del curso.

## Instrucciones

1. Asegurate de estar en la rama `main`.
2. Crea una rama nueva con tu nombre de usuario o nombre real:

git branch nombre-rama (no te moverá a la rama nueva, debes hacerlo manualmente)

O

git switch -c nombre-rama (creará tu rama y git apuntará a ella)

O en GitHub Desktop: Branch → New Branch.

Si creaste la rama con el comando git branch, luego apunta a la rama que creaste para seguir las tareas:
git switch nombre-rama

3. Realizá estos cambios:
- Crea un archivo nuevo en `colaboradores/` llamado `TU_NOMBRE.txt` y escribí algo sobre vos.
- Modificá alguno de los archivos dentro de `tareas/`, agregando una línea con tu nombre.

4. Hacé commit y subí la rama:

git push origin nombre-rama

5. Desde GitHub, creá un **Pull Request** hacia `main`.

6. Asigná el PR a otro colaborador para revisión.

7. Una vez aprobado, realizá el merge (o pedile a alguien que lo haga).
